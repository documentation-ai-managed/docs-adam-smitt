## Overview

Mott provides comprehensive progress tracking to help you monitor your Chechen language journey. Track badges, review history, set goals, and export reports to stay motivated. Access these features from your dashboard at `https://dashboard.mott.com/progress`.

## Viewing Achievement Badges and Milestones

Navigate to the Progress tab on your dashboard to see badges earned for milestones like completing 10 lessons or scoring `>90%` on quizzes.

<Expandable title="Badge Types" default-open="true">
  - **Beginner**: Complete first 5 lessons
  - **Conversational**: Pass 20 quizzes
  - **Fluent**: Achieve 100-day streak
</Expandable>

Use the milestone tracker to visualize progress toward your next badge.

```javascript
// Example API call to fetch badges
const response = await fetch('https://api.mott.com/v1/badges', {
  headers: { Authorization: `Bearer ${YOUR_TOKEN}` }
});
const badges = await response.json();
console.log(badges); // { earned: 3, next: "Conversational" }
```

## Setting Custom Goals and Reminders

Define daily or weekly goals to maintain consistency.

<Steps>
  <Step title="Create Goal" icon="plus">
    Go to **Progress** > **Goals** and click **New Goal**.
  </Step>
  <Step title="Configure Details">
    Set target (e.g., 30 minutes/day) and enable reminders.
  </Step>
  <Step title="Activate Reminders">
    Choose email or app notifications for missed days.
  </Step>
</Steps>

## Exporting Progress Reports

Export data in CSV or PDF for personal records or sharing.

<CodeGroup tabs="JavaScript,Python,cURL">
  ```javascript
  const response = await fetch('https://api.mott.com/v1/progress/export?format=csv', {
    method: 'GET',
    headers: {
      Authorization: `Bearer ${YOUR_TOKEN}`,
      'Content-Type': 'application/json'
    }
  });
  const data = await response.blob();
  const url = window.URL.createObjectURL(data);
  const a = document.createElement('a');
  a.href = url;
  a.download = 'mott-progress.csv';
  a.click();
  ```
  ```python
  import requests

  response = requests.get(
      'https://api.mott.com/v1/progress/export?format=csv',
      headers={'Authorization': f'Bearer {YOUR_TOKEN}'}
  )
  with open('mott-progress.csv', 'wb') as f:
      f.write(response.content)
  ```
  ```bash
  curl -H "Authorization: Bearer YOUR_TOKEN" \
       "https://api.mott.com/v1/progress/export?format=csv" \
       -o mott-progress.csv
  ```
</CodeGroup>

<ParamField path="format" param-type="string" required="true">
  Export type: `csv` or `pdf`.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token from your account settings.
</ParamField>
