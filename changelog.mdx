## Recent Releases

Mott releases updates regularly to improve your Chechen language learning journey. You receive automatic updates through the web app, or check manually in your dashboard settings.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added mobile responsiveness: Access lessons seamlessly on tablets and phones with adaptive layouts.
- Introduced interactive quizzes for vocabulary: Test your knowledge with real-time feedback after each lesson.

## Improvements

- Enhanced audio playback: Smoother pronunciation guides with adjustable speed controls.
- Updated lesson content: Added 20 new beginner phrases focused on greetings and daily conversations.

## Bug Fixes

- Fixed lesson progress syncing issues across devices.
- Resolved audio loading delays on slower connections.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["bugfix", "improvement"]}>

- Better search functionality: Now includes phonetic search for Chechen words.

- Fixed profile avatar upload errors.
- Patched notification delivery failures for daily streaks.

## How to Update

Keep Mott current to access the latest lessons and fixes.

<Steps>
  <Step title="Check for Updates" icon="refresh-cw">
    Open your dashboard at `https://dashboard.mott.com`.
    
    Navigate to Settings {">"} Updates.
  </Step>
  <Step title="Install Update" icon="download">
    Click "Update Now" if available.
    
````bash
# Example CLI update for desktop app (if applicable)
npm update @mott/desktop-app
````
  </Step>
  <Step title="Restart App" icon="zap">
    Restart Mott to apply changes and enjoy new features.
  </Step>
</Steps>

<Callout kind="tip">
  Enable auto-updates in settings to stay current without manual checks.
</Callout>
